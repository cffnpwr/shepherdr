# UI

## トレイメニュー

メニューバーのトレイアイコンから以下を提供する。

- サービスごとの項目
  - 状態表示（実行中 / 停止 / 再起動待ち / 失敗）
  - 起動 / 停止 / 再起動
- 設定を再読み込み（[サービス管理](./service-management.md)の差分適用）
- ログを開く（ログウィンドウの表示）
- Shepherdrを終了

## ログウィンドウ

- サービスを選び、そのログファイルをtail表示する（追記に追従する）。
- 検索・フィルタ・ローテーション済み世代の閲覧は持たない。

## フロントエンド構成

ログウィンドウはTauriのWebViewで実装する。

- ランタイム・パッケージ管理: bun
- 言語・ビルド: TypeScript＋Vite
- UI: React＋shadcn/ui（Tailwind CSS）

## デザイン方針

- ビジュアルデザインはmacOS 27 UI Kit（Apple Design Resources）に準拠させる。
- shadcn/uiのコンポーネントに、UI Kit準拠のテーマ（フォント・配色・コントロール形状）を当てて使う。
